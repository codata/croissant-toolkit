# 📨 Communication Officer
## Secure Delivery & SMTP Notifications

- **Secure Delivery**: Uses SMTP with TLS for secure email transmission.
- **Attachments**: Supports sending generated `.jsonld` files as attachments.
- **Flexible Configuration**: Configurable via environment variables for easy setup.

---

# 🥐 Croissant Expert
## MLCommons Metadata Specification

- **Core Logic**: Responsible for the [MLCommons Croissant](https://mlcommons.org/croissant) specification.
- **Standardization**: Transforms dataset descriptions into 100% compliant JSON-LD metadata.
- **Validation**: Ensures metadata integrity for ML model training.

---

# 🧭 Navigator
## Browser Automation & Web Scraping

- **Automation**: Opens Google Chrome or Firefox to perform searches and scraping.
- **Efficiency**: Automatically handles consent buttons and parses search result URLs.
- **Data Collection**: Saves results to `google_search_results.json` for further use.

---

# 🧠 NLP Expert
## Advanced Named Entity Recognition

- **Semantic Extraction**: Uses Gemini 3 to perform advanced Named Entity Recognition (NER).
- **Mapping**: Identifies key entities and maps them to standard Schema.org types in JSON-LD.
- **Contextual Understanding**: Extracts entities from descriptions, transcripts, and web results.

---

# 📓 Obsidian Expert
## Dataset to Knowledge Base Conversion

- **Knowledge Management**: Transforms Croissant JSON-LD files into interlinked Markdown notes.
- **Structured Notes**: Includes summaries, metadata, and technical JSON-LD sources.
- **Persistent Intelligence**: Creates a visual knowledge base of discovered datasets.

---

# 📸 Screenshot Taker
## Visual Verification & Archiving

- **Visual Snapshots**: Captures high-quality snapshots of web pages.
- **Multimodal Support**: Provides visual context for Gemini's multimodal reasoning.
- **Smart Waiting**: Automatically waits for the network to be idle before capturing.

---

# 📺 YouTuber
## Video Content Discovery & Analysis

- **Search YouTube**: Opens Google Chrome to search and extract video metadata.
- **Structured Data**: Extracts titles, URLs, and descriptions into JSON format.
- **Integration**: Feeds into the Transcriber for deep content analysis.

---

# ✍️ Transcriber
## YouTube Transcription & Analysis

- **Text Extraction**: Downloads text transcripts (closed captions) from YouTube videos.
- **Searchability**: Converts video content into searchable and indexable text.
- **Core Engine**: Essential for converting video tutorials into Croissant metadata.

---

# 🌐 Translator
## High-Precision Multilingual Support

- **Language Recognition**: Leverages Gemini 3 to identify the source language of any text.
- **Accuracy**: Provides high-precision translation into English for global discovery.
- **Seamless Integration**: Supports translation for transcripts, documents, and web content.

---

# 🧙 Wizard
## The Ultimate Data Integrator

- **Orchestration**: Combines transcription, translation, and NLP into a single pipeline.
- **Automation**: Fully automates the journey from raw discovery to Croissant serialization.
- **Efficiency**: Designed for high-speed metadata generation and dataset preparation.

---

# 🚶 Walker
## Recursive Web Exploration

- **Deep Crawling**: Recursively walks through web directories and links.
- **Discovery**: Uncovers hidden datasets and resources on complex websites.
- **Automated Mapping**: Builds a map of available resources for further extraction.

# 🕸️ Neo4j Expert

- **Graph Storage**: Ingests discovered metadata into Neo4j graph databases.
- **Relational Discovery**: Uncovers deep relationships between different datasets.
- **Advanced Querying**: Allows for complex semantic queries across the entire knowledge base.

# 🧪 Tester & Test-All
