# Tester Skill

This skill provides a complete suite for verifying the functionality of all skills in the Croissant Toolkit.

## Usage

### Run Navigator Test
```bash
python3 .gemini/skills/tester/scripts/test_navigator.py
```

### Run Youtuber Test

### Run Transcriber Test
