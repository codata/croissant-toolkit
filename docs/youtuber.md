# Youtuber

Search for videos on YouTube based on specific keywords. Get list of videos with title, description, and URL.

**Core functionality:** Opens Google Chrome to the YouTube search results and extracts video metadata (title, URL, description) to `youtube_search_results.json`.

## Usage

To use this skill, simply ask the AI agent:
```text
Use the Youtuber skill to search for <KEYWORDS> on YouTube.
```

## Example

```text
Use the Youtuber skill to search for "MLCommons Croissant introduction" on YouTube.
```
